Creating A Webpage Using Eclipse

Robert Tucker



The following webpages will demonstrate how to get started in HTML using Eclipse.




STEP 1

Once you open the Eclipse program, you need to be sure you are in the HTML(web) persepctive. Click on the icon in the upper righthand corner (see red arrow). A window will open up and you will see a menu of perspectives. Scroll down to the "Web" perspective, click on it and click "OK".
